# Project_Management_SQL
# Project Management System (SQL)

## Objective
- Manage Employees, Projects, Tasks, and Task Assignments.
- Analyze project progress with SQL queries.
- Use stored procedures and triggers to automate assignments and logging.

## Structure
- `Employees` table: employee details
- `Projects` table: project details
- `Tasks` table: task details
- `Task_Assignments` table: tasks assigned to employees
- `Task_Update_Log` table: logs task status updates

## Queries
- View tasks per project
- Count total/completed tasks
- Employee workload
- Overdue tasks

## Features
- Stored Procedure: Assign_Task
- Trigger: Task_Update_Trigger
